# Underdog

从`Prism`项目中移植出来Region、MVVM、Dialog、Modularity等功能，使用官方依赖注入容器的库。

此项目是根据`Prism`和`StupidBear`项目更改的，技术有限，改的很失败，所以用`Underdog`命名/(ㄒoㄒ)/~~
